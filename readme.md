## About LaraVue

LaraVue is an open source admin panel project built with Laravel 5.4 and Vue 2.js.

## Features

- rtl support

- Fetach Sidebar items from server

## Installation

Download this repo.

Run the following commands:

<pre>

composer install

npm install

npm run dev

php artisan serve

</pre>
